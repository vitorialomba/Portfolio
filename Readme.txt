# Portfólio Vitória França

Este é o portfólio online de Vitória França, uma desenvolvedora web em formação. O site é construído com as tecnologias HTML, CSS, JavaScript, Bootstrap, e outros recursos para criar uma apresentação pessoal, incluindo habilidades, projetos e informações de contato.

## Tecnologias Utilizadas

- **HTML**: Estruturação das páginas.
- **CSS**: Estilos para as páginas, garantindo uma apresentação moderna e responsiva.
- **JavaScript**: Funcionalidades interativas, como animações de texto.
- **Bootstrap**: Framework para garantir um design responsivo e layout eficiente.
- **Google Fonts**: Fonte personalizada para o design do site.
- **AOS (Animate On Scroll)**: Animações ao rolar a página.
- **Swiper**: Para criar slides interativos.

## Estrutura do Projeto

O projeto é organizado da seguinte forma:
/assets /css - main.css # Estilos principais /img - my-profile-img.jpeg # Imagem do perfil /vendor - bootstrap - bootstrap-icons - aos - glightbox - swiper index.html # Página inicial do portfólio
